## STRING004 - Sử dụng StringBuffer để lấy chiều dài chuỗi

### Đề bài

Cho một chuỗi`s`, hãy sử dụng Java  StringBuffer để in ra chiều dài của chuỗi`s`đó

Ví dụ:

Dữ liệu vào:

`s = ContentWriter`

Dữ liệu ra:

`13`

### Dữ liệu vào

- `0 < s.length < 100`

### Dữ liệu ra

- Mỗi dòng ứng với 1 test case hiển thị chiều dài của chuỗi

### Ví dụ

Input #1 
```
10
CJTq
f6N41NBaB
X8X
J1iyam5C
F
8nl
CcysHj
KpwZe0m
B6
fcmv7YxFIU
```


Output #1 
```
4
9
3
8
1
3
6
7
2
10
```
